# Blinkit Data Analysis using Python 📊

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the Blinkit dataset using **Python**.  
The notebook covers:
- Data cleaning and preprocessing  
- Summary statistics  
- Data visualization  
- Insights into sales, items, and outlets  

It helps in understanding customer purchase patterns and business performance for Blinkit.  

---
Technologies Used

Python 🐍


:Pandas – Data manipulation

:NumPy – Numerical computing

:Matplotlib & Seaborn – Data visualization

:Jupyter Notebook – Interactive environment
